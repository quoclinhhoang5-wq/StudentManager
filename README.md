Student Manager Project

**1. Mô tả ngắn về project
Đây là phần mềm Quản lý sinh viên đơn giản được viết bằng ngôn ngữ C++.
Chức năng chính bao gồm:
- Thêm sinh viên mới.
- Hiển thị danh sách sinh viên.
- (Ghi thêm chức năng khác nếu có, ví dụ: Xóa, Sửa...).

**2. Hướng dẫn chạy
Để chạy chương trình, thực hiện các bước sau:
1. Clone repository về máy: git clone https://github.com/quoclinhhoang5-wq/StudentManager.git
2. Mở project bằng IDE (Visual Studio, Dev-C++, hoặc VS Code).
3. Build và Run file 

**3. Danh sách 5 commit chính
1. Initial commit: Khởi tạo dự án.
2. Add Student class: Thêm lớp đối tượng sinh viên.
3. Implement Add function: Viết hàm thêm sinh viên.
4. Update README: Cập nhật hướng dẫn sử dụng.
5. Merge branch feature/intro: Hợp nhất tính năng giới thiệu.
